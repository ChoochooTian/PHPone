# 1.PHP代码写在哪儿？
Sublime Text 3、JetBrains PhpStorm

# 2.PHP文件放到哪儿？
PHP程序文件需要放在服务器的网站根目录下

# 3.相关问题
- PHP程序文件的扩展名为.php
- PHP程序可以嵌入到HTML中
- PHP程序必须放到一对PHP标记中，才能解释为PHP程序
- PHP中每一条语句以分号结束
- PHP程序结构同C语言非常相似，它是由 C 和 Perl 语言发展而来
- PHP是弱类型语言

# 4.变量、常量、数据类型  
定义变量
- 不需要声明，直接赋值创建
- 以“$”符号开头，标识符区分大小写
- 定义变量时不需要指定变量的数据类型
- 变量定义后，还可以存储任意类型的数据
